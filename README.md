# Student Management System

Full-stack student management application with a Spring Boot backend and a React (Vite) frontend.

## Features

- Admin, Teacher, and Student management
- Authentication and role-based access
- Student-teacher assignment
- Search and pagination
- Responsive UI

## Tech Stack

### Backend
- Java, Spring Boot, Spring Security
- JPA/Hibernate
- Maven

### Frontend
- React + Vite
- Tailwind CSS
- Axios

## Project Structure

- backend/ — Spring Boot API
- frontend/ — React (Vite) UI

## Run Locally

### Backend

```bash
cd backend
./mvnw spring-boot:run
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```

## Configuration

- Update the frontend API base URL if deploying the backend separately.
