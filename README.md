# Student Management System

Full-stack student management app with a Spring Boot backend and a React (Vite) frontend.

## Live Demo
- Login: https://student-management-system-648n.vercel.app/login

## Project Structure
- backend/ — Spring Boot API
- frontend/ — React (Vite) UI

## Run Locally

### Backend
```bash
cd backend
./mvnw spring-boot:run
```

### Frontend
```bash
cd frontend
npm install
npm run dev
```

## Notes
- Configure backend URL for production deployments.
